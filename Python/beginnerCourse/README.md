# Rock Paper Scissors Lizard Spock

This project is a console-based game developed as part of a Python checkpoint project on Codédex.

In this mini-game, you play Rock Paper Scissors Lizard Spock against the computer and try to beat the CPU.


## 🎮 Description

Rock Paper Scissors Lizard Spock is an extended version of the classic game with two additional choices: Lizard and Spock.

You play against the computer by choosing one of five options:

* ✊ Rock
* ✋ Paper
* ✌️ Scissors
* 🦎 Lizard
* 🖖 Spock

The computer randomly picks its own choice, and the game decides the winner based on the classic rules of Rock Paper Scissors Lizard Spock.

## 🎮 Features
* Interactive console input using input()
* Random computer choice using random.randint()
* Extended game rules (Rock, Paper, Scissors, Lizard, Spock)
* Clear win/lose/tie logic
* Simple and replayable game loop

## Rules

* Rock crushes Scissors and Lizard
* Paper covers Rock and disproves Spock
* Scissors cuts Paper and decapitates Lizard
* Lizard eats Paper and poisons Spock
* Spock smashes Scissors and vaporizes Rock

## How to Run

* Run the Python file
* Choose a number from 1 to 5:
* The computer will randomly choose its move
* The result (win, lose, or tie) will be displayed
