This project is a graphical Magic 8 Ball application developed as part of a Java final project.

In this application, the user can ask questions and receive mystical, randomly generated answers from a magical oracle.

## 📖 Description

The Magic 8 Ball is a mysterious oracle that reveals your destiny… or keeps it hidden.

Simply type your question into the input field, press the Ask button, and let the sphere decide your fate ✨.

The application features a custom-designed interface with a magical background, dynamic text scaling, and atmospheric responses.

## 🎮 Features

* Interactive GUI built with Java Swing
* Custom background image with responsive scaling
* Custom font integration for a magical look 
* Dynamic UI that adapts to window size
* Randomized mystical answers
* Automatic answer fade (disappears after a few seconds)

## 🔮 Answer System

The oracle responds with different types of answers:

🟢 Positive
 
   * "The stars align in your favor."
   * "Yes... it is destined."
   * "The universe agrees."

🔴 Negative
 
 * "Darkness says no."
 * "The answer is forbidden."
 * "It shall not be."
 
 🟡 Unclear
 
 * "The fog is too thick..."
 * "Moonlight will tell."
 * "The answer is unclear."

Each answer is randomly selected to create an unpredictable experience.

## How to Use
1. Enter your question into the input field
2. Click the Ask button
3. Receive your answer from the oracle
4. The answer will disappear after a short time
